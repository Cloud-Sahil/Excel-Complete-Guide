## MODULE 2 — Excel Learning Roadmap

```
  EXCEL LEARNING PATH
  ════════════════════════════════════════════════════════
  
  🟢 BASICS           🟡 INTERMEDIATE       🔴 ADVANCED
  ┌──────────────┐    ┌──────────────────┐   ┌──────────────────┐
  │ Workbook     │ →  │ VLOOKUP/INDEX    │ → │ Power Query      │
  │ Navigation   │    │ MATCH            │   │ Power Pivot      │
  │ Data Entry   │    │ IF/IFS/NESTED    │   │ Macros/VBA       │
  │ Basic Format │    │ PivotTables      │   │ Dynamic Arrays   │
  │ SUM/AVG/COUNT│    │ Charts           │   │ LAMBDA           │
  │ Sort/Filter  │    │ Data Validation  │   │ Dashboards       │
  │ Basic Charts │    │ Conditional Fmt  │   │ Power BI Prep    │
  └──────────────┘    └──────────────────┘   └──────────────────┘
       Week 1-2            Week 3-6               Week 7-12
```

### 🟢 BASICS — Week 1 to 2

#### What to Learn

| Topic | Skills | Time |
|-------|--------|------|
| **Excel Interface** | Ribbon, Quick Access Toolbar, Name Box, Formula Bar | Day 1 |
| **Workbook & Sheets** | Create, rename, delete, color, move sheets | Day 1 |
| **Data Entry** | Text, numbers, dates, special characters | Day 2 |
| **Navigation** | Keyboard shortcuts, freeze panes, split screen | Day 2 |
| **Basic Formatting** | Font, fill, borders, alignment, number format | Day 3 |
| **Basic Formulas** | SUM, AVERAGE, COUNT, MIN, MAX | Day 4 |
| **Sort & Filter** | Single/multi-column sort, AutoFilter | Day 5 |
| **Basic Charts** | Bar, line, pie chart creation | Day 6-7 |
| **Print Setup** | Page layout, headers/footers, print area | Day 8 |

#### Beginner Checklist
```
□ Enter data in cells (text, numbers, dates)
□ Format cells (bold, color, borders)
□ Write SUM, AVERAGE, COUNT formulas
□ Use AutoFill to copy formulas
□ Apply sort and basic filters
□ Create a simple bar/pie chart
□ Save as .xlsx and .pdf
```

### 🟡 INTERMEDIATE — Week 3 to 6

#### What to Learn

| Topic | Skills | Time |
|-------|--------|------|
| **VLOOKUP / XLOOKUP** | Find values across tables | Week 3 |

| **Conditional Formatting** | Color scales, data bars, icon sets, rules | Week 5 |
| **Data Validation** | Dropdowns, input restrictions | Week 5-6 |
| **Named Ranges** | Create and use named ranges in formulas | Week 6 |
| **Intermediate Charts** | Combo charts, secondary axis, sparklines | Week 6 |

#### Intermediate Checklist
```
□ Write VLOOKUP with exact and approximate match
□ Use XLOOKUP (modern replacement for VLOOKUP)
□ Build nested IF formulas (3+ levels)
□ Use INDEX+MATCH for flexible lookups
□ Apply SUMIF/COUNTIF with multiple criteria
□ Clean data using TRIM, LEFT, RIGHT, MID
□ Build and format a PivotTable
□ Create conditional formatting rules
□ Set up a drop-down list with data validation
```

## MODULE 12 — Excel Formula Writing Guide

### Formula Anatomy

```
  = SUMIFS ( C2:C100 , A2:A100 , "Sales" , B2:B100 , "Q1" )
  ▲ ─────── ──────── ──────────────────────────────────────
  │ Function Sum Range  Criteria Range1   Criteria1   Range2  Criteria2
  │
  └─ Every formula starts with =
```

### Formula Writing Rules

| Rule | Description | Example |
|------|-------------|---------|
| **Always start with =** | Required prefix | `=SUM(A1:A10)` |
| **Use absolute refs ($)** | Lock range in formula | `=$B$2` (F4 key) |
| **Avoid hardcoding values** | Use cell references | `=A2*B2` not `=A2*1.18` |
| **Name your ranges** | More readable formulas | `=SUM(SalesQ1)` |
| **Use IFERROR** | Handle errors gracefully | `=IFERROR(A1/B1,0)` |
| **Document complex formulas** | Add a comment column | Use a helper column |

### Absolute vs Relative References

```
  A1    → Relative (both row and column change when copied)
  $A1   → Column absolute (column stays, row changes)
  A$1   → Row absolute (row stays, column changes)
  $A$1  → Fully absolute (nothing changes when copied)

  Use F4 to toggle between: A1 → $A$1 → A$1 → $A1 → A1

  When to use absolute reference:
  ✅ Lookup table range: =VLOOKUP(A2, $F$2:$H$100, 2, 0)
  ✅ Fixed rate/tax: =B2 * $J$1  (J1 has the tax rate)
  ✅ SUMIF criteria range: =SUMIF($A:$A, D2, $B:$B)
```

### Nested Formula Pattern

```excel
// Build complex formulas step by step:

// Step 1: Simple lookup
=VLOOKUP(A2, Products, 2, 0)

// Step 2: Add error handling
=IFERROR(VLOOKUP(A2, Products, 2, 0), "Not Found")

// Step 3: Add conditional logic
=IF(A2="", "", IFERROR(VLOOKUP(A2, Products, 2, 0), "Not Found"))

// Step 4: Add text formatting
=IF(A2="", "", IFERROR(PROPER(VLOOKUP(A2, Products, 2, 0)), "Not Found"))
```

### Formula Debugging Tips

```
Technique 1: F9 to evaluate parts
  Select part of formula in formula bar → press F9 → see value
  Press Escape to restore formula

Technique 2: Show all formulas (Ctrl + ~)
  Toggle between values and formulas view

Technique 3: Evaluate Formula dialog
  Formulas tab → Evaluate Formula → Step through calculation

Technique 4: Trace Precedents
  Formulas tab → Trace Precedents → see which cells feed this formula

Technique 5: Error checking
  Formulas tab → Error Checking → find all formula errors

Common errors:
  #N/A    → Value not found (VLOOKUP/MATCH)
  #VALUE! → Wrong data type (text where number expected)
  #REF!   → Invalid reference (deleted row/column)
  #DIV/0! → Division by zero (use IFERROR or IF(B1=0,0,A1/B1))
  #NAME?  → Function misspelled or name not defined
  #NUM!   → Invalid number (sqrt of negative)
  ######  → Column too narrow (widen the column)
```

### Formula Writing Cheat Sheet

```
FORMULA COMPONENTS:
══════════════════════════════════════════════════════════════

  Function:    =FUNCTION(argument1, argument2, ...)
  Arithmetic:  =A1+B1  /  =A1-B1  /  =A1*B1  /  =A1/B1
  Comparison:  =A1>B1  /  =A1<B1  /  =A1>=B1  /  =A1<>B1
  Text join:   =A1&" "&B1  or  =CONCAT(A1," ",B1)
  Array:       =FILTER(A:B, B:B>50)  (spills automatically)

REFERENCE TYPES:
══════════════════════════════════════════════════════════════
  A1           → relative (changes on copy)
  $A$1         → absolute (never changes)
  $A1          → column locked only
  A$1          → row locked only
  A1:B10       → range
  A:A          → entire column
  1:1          → entire row
  Sheet2!A1    → another sheet
  [Book2]Sheet1!A1  → another workbook

COMMON PATTERNS:
══════════════════════════════════════════════════════════════
  Conditional sum:    =SUMIFS(sum, range1, crit1, range2, crit2)
  Safe lookup:        =IFERROR(XLOOKUP(A2,B:B,C:C), "N/A")
  Rank with ties:     =RANK(B2,$B$2:$B$100,0)
  % of total:         =B2/SUM($B$2:$B$100)
  Running total:      =SUM($B$2:B2)
  YOY change:         =(B2-A2)/A2
  Dynamic header:     ="Sales — "&TEXT(TODAY(),"MMM YYYY")
  Conditional color:  (use Conditional Formatting, not formula)
```
